# Professional Portfolio - Shubham Kumar Bal

A cutting-edge, professional portfolio website built with modern web technologies featuring advanced animations, 3D effects, and premium design.

## 🚀 Features

### 🎨 Design
- **Glass Morphism** - Modern glass effect design
- **Dark/Light Theme** - Full theme switching capability
- **Responsive Design** - Fully responsive across all devices
- **Premium Typography** - Professional font hierarchy and spacing
- **Gradient System** - Custom gradient system for visual appeal

### ⚡ Animations & Effects
- **Three.js 3D Scene** - Interactive 3D background with floating geometry
- **Particle System** - Dynamic particle system with mouse interaction
- **Advanced Scroll Animations** - Smooth scroll-triggered animations
- **Custom Cursor** - Magnetic cursor with hover effects
- **Magnetic Buttons** - Interactive buttons with magnetic pull effect
- **Ripple Effects** - Click ripple animations
- **Text Reveal** - Typing and text reveal animations
- **Parallax Scrolling** - Depth effect on scroll

### 🛠️ Technical Features
- **Modular JavaScript** - ES6 classes with clean architecture
- **CSS Custom Properties** - Theming and design system
- **Intersection Observer** - Efficient scroll animations
- **Web Audio API** - Interactive sound effects
- **RequestAnimationFrame** - Smooth 60fps animations
- **Progressive Enhancement** - Graceful degradation
